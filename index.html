Container

Instanz eines Images.

Image

Fertige Umgebung, in der Code ausgeführt werden kann. Besteht aus (mehreren) Layern.

Layer

Eine Instruktion zur Erstellung eines Images.

https://www.crisp-research.com/docker-container-die-zukunft-moderner-applikationen-und-multi-cloud-deployments/

Vorteile

  • Portabilität von Anwendungen
    • Lösung von Anwendung und Infrastruktur
    • weniger Abhängigkeiten
  • Einfacher
    • man muss kein Linux-Profi sein
  • Weniger Sicherheitslücken
    • Minimale Images haben nur die notwendige Software
    • ... aber Kernel wird geteilt
